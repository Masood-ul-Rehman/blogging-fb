- ‚úÖ `convex/facebook/mutations.ts` - 4 mutations

  - `updateAdAccounts` - Sync ad accounts

- ‚úÖ `convex/facebook/queries.ts` - 4 queries

  - `getAdAccounts` - List ad accounts
  - `getActionLogs` - View audit history

- ‚úÖ `convex/facebook/actions.ts` - 5 server-side actions

  - `fetchAdAccountInsights` - Get performance data
  - `fetchCampaigns` - List campaigns

  - `updateAdStatus` - Pause/resume ads

```
User ‚Üí React Component ‚Üí Convex Query/Action ‚Üí Facebook Graph API
                          ‚Üë
                     (Server-Side Only)
```

### Security Model

```
Client
  ‚îú‚îÄ‚îÄ Can: View connection status (without token)
  ‚îú‚îÄ‚îÄ Can: View ad accounts list
  ‚îú‚îÄ‚îÄ Can: Trigger server actions
  ‚îî‚îÄ‚îÄ Cannot: Access tokens or make direct FB API calls

  ‚îú‚îÄ‚îÄ Stores: Encrypted tokens
```
